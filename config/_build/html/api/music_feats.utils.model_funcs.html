STA¶

music_feats.utils.model_funcs.STA(delRstim, zRresp, delPstims, zPresps, ndelays, modeldims)¶

Function estimates model weights using a spike-triggered average model. Weights are cross-validated and the results returned.

Parameters:

delRstim : np.ndarray

training - stimulus feature time series

zRresp : np.ndarray

training - voxel responses

delPstims : list of np.ndarry

validation - stimulus feature time series

zPresps : list of np.ndarray

validation - voxel responses

ndelays : python list of ints

number of delays to use in the model

modeldims : python list of ints,

dimensions of the submodels as a list
Returns:

np.ndarray, np.ndarray, list, list

STA_W : estimated weights, valcorr_STA : validation correlation values (full model) submodelwts_STA : wts for each submodel, valmodelcorr_STA : list of prediction correlcation values for each submodel

averageDelays¶

music_feats.utils.model_funcs.averageDelays(wt, modeldims, delays=4)¶

Compute the average weight across delays. Assumes there are wt.shape[0]/delays features overall (sub-models of dimension 1). Used for tonotopy analysis.

Parameters:

wt : np.ndarray

[(num_channels*delays, num_voxels)]. Delayed model weights.

modeldims: list

List of submodel dimensions (should be list of all 1s).

delays : int

Number of delays used. Default: 4.
Returns:

int

Average of delays

separate_model_weights¶

music_feats.utils.model_funcs.separate_model_weights(vwts, ndelays, modeldims)¶

Takes an N x M matrix of voxel weights [vwts] (where N is the total number of features and M is the number of voxels) and separates out the weights for each separate model.

Parameters:

vwt : np.ndarray

weight vector

ndelays : int

number of delays

modeldims : modeldims
Returns:

np.ndarray

List of (D*Ni) x M matrices, where D is the number of

delays and Ni is the number of features in model i.

spectralCentroid¶

music_feats.utils.model_funcs.spectralCentroid(W)¶

Function used for calculating spectral centroid of the weights. Computes the centroid of the weights (rows) for each voxel (column). Frequency approximated by index of the weights.

Parameters:

W : np.ndarray

[(num_channels, num_voxels)].
Returns:

np,ndarray,

SC :[(num_voxels)] One value per voxel correspond to the centroid of frequency indices.
