separate_model_weights¶

music_feats.utils.viz.separate_model_weights(vwts, ndelays, modeldims)¶

Takes an N x M matrix of voxel weights [vwts] (where N is the total number of features and M is the number of voxels) and separates out the weights for each separate model.

Parameters:

vwt : np.ndarray

weight vector

ndelays : int

number of delays

modeldims : modeldims
Returns:

np.ndarray

List of (D*Ni) x M matrices, where D is the number of

delays and Ni is the number of features in model i.

separate_voxel_model_weights¶

music_feats.utils.viz.separate_voxel_model_weights(vwt, ndelays, modeldims)¶

Takes a voxel weight vector [vwt] and separates out the weights for each model. Returns a list of DxNi matrices, where D is [ndelays] and Ni is the number of dimensions in model i, given in [modeldims].

Parameters:

vwt : np.ndarray

weight vector

ndelays : int

number of delays

modeldims : modeldims
Returns:

np.ndarray

List of DxNi matrices
